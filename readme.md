# **Budget tracker app** 
>
## **Description**

Budget tracker is a simple Python application that helps users track their finances. It automatically handles fixed expenses and provides information about the remaining budget for the current month. This makes it useful for household financial planning and saving for unexpected expenses in the future.

### *Features:*

 - Easily track income and expenses 
 - Edit and delete entries
 - Automatically add regular expenses 
 - View income and expenses for a selected period
 - See the remaining budget for the current month
 - Help users save money
